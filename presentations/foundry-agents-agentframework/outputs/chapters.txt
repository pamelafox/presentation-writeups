This session is the first in a three-part series on hosting agents on Microsoft Foundry. Pamela Fox walks through building Python agents using the Microsoft Agent Framework (MAF), progressively enhancing them with Foundry models, Foundry IQ knowledge retrieval, and the Foundry Toolbox. She then shows how to host those agents (and multi-agent workflows) on Foundry Agent Service using the Responses API adapter, `azd up` deployments, and OpenTelemetry observability.
