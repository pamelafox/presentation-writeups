This talk covers how to build a Retrieval Augmented Generation (RAG) chat application on Azure. It introduces large language models, explains RAG and the importance of hybrid search, walks through an open-source RAG chat app solution with Azure AI Search and OpenAI, and discusses evaluation and observability techniques for RAG applications.
