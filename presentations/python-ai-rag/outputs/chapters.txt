This session covers Retrieval Augmented Generation (RAG), a technique for giving LLMs access to external data so they can provide grounded, accurate answers. The talk walks through RAG fundamentals, progressively more advanced RAG flows in Python, document ingestion and hybrid search, and full end-to-end RAG applications on PostgreSQL and Azure AI Search.
