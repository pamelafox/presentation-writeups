This video covers Foundry IQ knowledge bases, which unify multiple knowledge sources behind a single endpoint for agentic retrieval. Pamela Fox and Matthew Gotteiner walk through the architecture (query planning, knowledge sources, output merging), the three retrieval reasoning effort levels (minimal, low, medium), and demonstrate multi-source queries using an open-source sample app, Foundry Agents, and GitHub Copilot via MCP.
