In the final session of the Python + Agents series, Pamela Fox explores how to incorporate human-in-the-loop (HITL) interactions into agentic workflows using the Microsoft Agent Framework. The session covers tool approval for gating sensitive operations, structured requests and responses for collecting human input, checkpoint storage for durable long-running workflows, and handoff orchestration with interactive user input. The session concludes with a full-stack banking assistant demo that combines all these patterns.
