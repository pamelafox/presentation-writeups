This session is the second in a three-part series on hosting Python agents on Microsoft Foundry. It covers building agents with LangChain and LangGraph, integrating them with Foundry Models, Foundry IQ, and Foundry Tools, hosting them on Foundry Agent Service using the Responses API adapter, adding observability with OpenTelemetry, and building multi-agent workflows with LangGraph.
