# Python + AI: RAG

Part of a 9-part series on Python+GenAI

https://www.youtube.com/watch?v=DHebceyjCbk

Slides: PythonAI-Oct-RAG.pdf

We explore Retrieval Augmented Generation (RAG), an approach that provides context to an LLM so it can deliver well-grounded answers for a particular domain. We show how RAG works with many data sources, walk through RAG flows in Python, and culminate in a full-stack RAG application based on Azure AI Search.
