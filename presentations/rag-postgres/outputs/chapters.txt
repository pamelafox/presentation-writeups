This video covers how to build Retrieval Augmented Generation (RAG) applications using PostgreSQL. Pamela Fox demonstrates setting up pgvector for vector search, full-text search, hybrid search with Reciprocal Rank Fusion, and orchestrating a complete RAG flow in Python. She then walks through a full RAG-on-PostgreSQL solution deployable to Azure. Joshua Johnson follows with a deep dive into Azure Database for PostgreSQL AI extensions, including the azure_ai extension for calling Azure OpenAI from SQL, in-database embedding models via azure_local_ai, and a demo combining vector search, geospatial queries, and translation services.
