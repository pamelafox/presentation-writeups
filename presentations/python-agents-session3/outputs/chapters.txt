This session covers monitoring and evaluating AI agents built with the Microsoft Agent Framework. It demonstrates how to use OpenTelemetry for observability—exporting traces, metrics, and logs to platforms like Aspire and Azure Application Insights. It then covers automated evaluation using the Azure AI Evaluation SDK, including tool call accuracy, intent resolution, task adherence, and response completeness evaluators. The session concludes with safety evaluation through automated red teaming.
