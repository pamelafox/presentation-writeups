This session covers building AI-driven workflows using the Microsoft Agent Framework in Python. It introduces the core concepts of workflows—executors, edges, messages, and outputs—then progressively builds from basic sequential workflows to conditional branching with structured outputs, state management, and a full-stack application demo.
