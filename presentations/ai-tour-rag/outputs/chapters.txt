This talk covers advanced retrieval strategies for AI apps and agents on Azure. It demonstrates optimal hybrid search (keyword + vector + reranking), graph-based retrieval with Azure Database for PostgreSQL using Apache AGE, and agentic retrieval with Azure AI Search knowledge bases.
